

## Description

EMR backend application.

## Installation

```bash
$ npm install
$ cp .env.example .env
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```

## Swagger

You can access a list of the available endpoints on http://localhost:3000/api#/
